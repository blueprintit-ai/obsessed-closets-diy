# Obsessed Closets DIY Landing Page

Professional React landing page for the Obsessed Closets DIY program.

## Quick Start

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

1. Upload to GitHub
2. Connect to Vercel
3. Deploy automatically

## Features

- React 18 + Vite
- Tailwind CSS
- Responsive design
- Contact form
- 5-step process
- Professional UI
