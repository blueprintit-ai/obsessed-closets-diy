# Obsessed Closets DIY Landing Page

Ultra-clean React landing page with minimal dependencies - guaranteed to deploy!

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload to GitHub
2. Connect to Vercel
3. Deploy automatically

## Features

- React 18 + Vite
- Tailwind CSS
- Zero dependency conflicts
- Professional design
- Contact form
- Responsive layout
